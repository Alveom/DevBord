Q: What are the different ways to select an element in the DOM?

Ans:Imagine your webpage is a tree, and every part of the page (like buttons, text, images) is a branch or leaf on that tree. The DOM (Document Object Model) is that tree, and JavaScript lets you "grab" these branches/leaves (elements) to change them (like changing colors, text, etc.).

Q: What is the difference between innerHTML, innerText, and textContent ?

Ans:What is innerHTML? → The HTML content inside an element (including tags).
What is innerText? → The visible text you see (ignores hidden text/HTML).
What is textContent? → All text (including hidden) but no HTML tags.

Q: What is event delegation in the DOM?

Ans: Letting a parent element handle events for its children so we don’t add event listeners to every kid.

Q: What is event bubbling in the DOM?

Ans: When you click something like a button, the click travels up to its parent like the div or page around it.

Q: How do you create, add, and remove elements using JavaScript?

Ans:Create an Element: document.createElement("tagName") → Makes a new element.
Add to Page: parentElement.appendChild(newElement) → Slaps it into the DOM (like adding to a list).
Remove an Element: element.remove() → Deletes it from the page.
